# SOC Log Analyzer

## Overview
The SOC Log Analyzer is a Python-based tool designed to simulate basic Security Operations Center (SOC) functionality. It processes log files, identifies potential security threats, classifies events by severity, and generates a structured incident report.

This project demonstrates foundational cybersecurity skills in log analysis, threat detection, and incident response, aligned with real-world SOC operations.

---

## Key Features
- Detection of failed login attempts  
- Identification of suspicious system activity  
- Severity classification of security events (Low, Medium, High)  
- Structured incident reporting output  
- Simple and extensible log analysis framework  

---

## Technologies Used
- Python  
- Regular Expressions (Regex)  
- Command Line Interface (CLI)  

---

## How It Works
1. The tool reads a log file containing system or network events  
2. It scans each entry for predefined suspicious patterns  
3. Identified events are classified based on severity  
4. A structured security report is generated for analysis  

---

## Sample Use Case
This tool can be used to:
- Simulate SOC alert monitoring  
- Practice log analysis and incident detection  
- Understand basic SIEM workflows  
- Build foundational cybersecurity skills
